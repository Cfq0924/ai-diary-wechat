## 使用步骤

### 步骤 1: 上传云函数

1. 打开微信开发者工具
2. 在左侧文件树找到 `cloudfunctions/generateTags` 目录
3. **右键点击** 该目录
4. 选择 **"上传并部署：云端安装依赖"**
5. 等待上传完成（约 30 秒）

### 步骤 2: 测试

1. 在小程序中写一篇新日记（至少 10 个字）
2. 点击保存
3. 打开微信开发者工具的 **控制台**
4. 应该能看到类似输出：
   ```
   生成的标签：["工作", "生活", "感悟"]
   ```
5. 查看首页，日记下方应该显示绿色标签

## 常见问题

**Q: 云函数找不到？**
A: 确保 `cloudfunctions/generateTags` 目录存在，并且已经上传

**Q: API 调用失败？**
A: 打开云开发控制台 → 云函数 → generateTags → 日志，查看错误信息

**Q: 标签生成慢？**
A: 首次调用会慢一些（冷启动），后续会快
