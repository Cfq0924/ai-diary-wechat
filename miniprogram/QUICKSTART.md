# 快速启动指南

## 3 分钟运行起来

### 步骤 1: 打开项目
用微信开发者工具导入 `miniprogram` 目录

### 步骤 2: 配置云开发
1. 点击顶部"云开发"按钮
2. 开通云开发（免费版即可）
3. 复制环境 ID（类似 cloud1-xxx）

### 步骤 3: 修改配置
编辑 `app.js` 第 7 行：
```javascript
env: 'your-env-id',  // 改成你的环境 ID
```

### 步骤 4: 创建数据库
1. 云开发控制台 → 数据库
2. 点击"+ 创建集合"
3. 输入集合名：`diary_entries`
4. 权限选择："所有用户可读写"

### 步骤 5: 部署云函数

右键以下云函数目录，上传并部署（云端安装依赖）：
- `cloudfunctions/generateTags`
- `cloudfunctions/generateEmbedding`
- `cloudfunctions/manageTags`

### 步骤 6: 编译运行
点击"编译"按钮

## 常见问题

### Q: 没有 AppID 怎么办？
A: 可以使用微信官方提供的"测试号"，或者在 project.config.json 中填入任意 AppID（仅用于本地调试，无法上传）

### Q: 云开发开通失败？
A: 确保你的微信账号已实名认证

### Q: 数据库创建后还是报错？
A: 检查集合名称是否正确（diary_entries，注意是复数）

### Q: TabBar 图标缺失？
A: 临时方案：在 app.json 中注释掉 tabBar 配置，或放入任意 81x81 的 PNG 图片
