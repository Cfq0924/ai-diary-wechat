# 图标资源说明

TabBar 需要以下图标（建议尺寸 81x81px）：

## 首页图标
- `home.png` - 灰色房子图标（未选中）
- `home-active.png` - 绿色房子图标（选中）

## 搜索图标
- `search.png` - 灰色放大镜图标（未选中）
- `search-active.png` - 绿色放大镜图标（选中）

## 空状态图标
- `empty.png` - 空状态插图（建议 240x240px）

---

你可以：
1. 使用设计工具（Figma、Sketch）自行设计
2. 从图标库下载（iconfont.cn、iconpark.cn）
3. 使用微信开发者工具自带的图标资源

临时方案：可以暂时注释掉 app.json 中 tabBar 配置，先不显示图标。
